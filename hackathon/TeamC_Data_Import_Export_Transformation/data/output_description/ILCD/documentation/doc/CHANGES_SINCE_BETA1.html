General/multiple schemas affected

  • renamed types with reference to "Conformity" to reflect new nomenclature ("Compliance")
  • removed redundant constraints
  • add common:referenceToOwnershipOfDataSet to publicationAndOwnership due to restructuring of PublicationAndOwnershipGroup1
  • added unique constraint for common:class/@level on classification
  • added unique constraint for common:category/@level on elementaryFlowCategorization

Common_EnumerationValues.xsd

  • added enumeration StandardMethodTypeValues
  • enumeration GlobalReferenceTypeValues: renamed entry "process or LCI result data set" to "process data set"
  • TypeOfReviewValues: renamed "Third party review (independent)" to "Accredited third party review"

ILCD_Common_Groups.xsd

  • renamed group ConsistencyAndConformityGroup1 to ComplianceGroup
  • removed group PublicationAndOwnership1Process (redundant, references changed to PublicationAndOwnershipGroup1 and common:referenceToOwnershipOfDataSet added)
  • renamed CategoryInformationType to ClassificationInformationType
  • removed global element categoryInformation (redundant)
  • renamed element referenceToEntitiesAndPersonsWithExclusiveAccessToThisDataSet to referenceToEntitiesWithExclusiveAccess

ILCD_ContactDataSet.xsd

  • elements contactAddress, centralContactPoint: changed maxoccurs to 100 (was 1) because of multilang type
  • added unique constraint for contact:contactAddress/@xml:lang
  • added unique constraint for contact:centralContactPoint/@xml:lang
  • added unique constraint for contact:contactDescriptionOrComment/@xml:lang

ILCD_FlowDataSet.xsd

  • removed redundant type LCIAFactorsType
  • removed flowPropertiesAndLCIAFactors container (but not contained element flowProperties)
  • added unique constraint for common:accessRestrictions/@xml:lang on publicationAndOwnership
  • fixed unique constraint for common:synonyms/@xml:lang on dataSetInformation
  • fixed unique constraint for common:generalComment/@xml:lang on dataSetInformation
  • fixed unique constraint for flow:locationOfSupply/@xml:lang on geography
  • added unique constraint for common:shortDescription/@xml:lang on referenceToTechnicalSpecification
  • fixed key/keyref for flow:flowProperty/@dataSetInternalID and flow:referenceToReferenceFlowProperty

ILCD_FlowPropertyDataSet.xsd

  • added unique constraint for common:shortDescription/@xml:lang on flowproperty:referenceToOwnershipOfDataSet
  • fixed unique constraint for common:accessRestrictions/@xml:lang on publicationAndOwnership
  • element referenceToDataSource on dataSourcesTreatmentAndRepresentativeness: added minOccurs="0"
  • fixed unique constraint for common:name/@xml:lang on dataSetInformation
  • fixed unique constraint for common:synonyms/@xml:lang on dataSetInformation
  • fixed unique constraint for common:generalComment/@xml:lang on dataSetInformation

ILCD_LCIAMethodDataSet.xsd

  • renamed element LCIAMethodFlowDiagrammOrPicture to referenceToLCIAMethodFlowDiagrammOrPicture, changed type to common:GlobalReferenceType
  • adjusted order of elements in dataSetInformation to reflect structure in other datasets
  • added new element methodology to dataSetInformation, refers to newly added MethodologyType
  • added unique constraint for lciamethod:referenceToFateAndEffectModel/@xml:lang on dataGenerator
  • added unique constraint for accessRestrictions/@xml:lang
  • fixed unique constraint for common:name/@xml:lang on dataSetInformation
  • fixed unique constraint for common:synonyms/@xml:lang on dataSetInformation
  • fixed unique constraint for common:generalComment/@xml:lang on dataSetInformation
  • fixed unique constraint for common:timeRepresentativenessDescription/@xml:lang on time
  • added unique constraint for lciamethod:LCIAMethodDescriptionAndIncludedSubMethods/@xml:lang on impactModel
  • fixed unique constraint for common:scope/@name on review
  • added unique constraint for common:reviewDetails/@xml:lang on review
  • added unique constraint for common:otherReviewDetails/@xml:lang on review
